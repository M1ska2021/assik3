## Project Overview
This project is a refactored version of **Assignment 3**, redesigned to fully comply with **SOLID principles** and to demonstrate **advanced Java OOP features**.

The domain of the project is a **Cartoon Management System**, which allows creating, reading, updating, and deleting cartoons using a layered architecture and a PostgreSQL database.

## Technologies Used
- Java (OOP, SOLID)
- JDBC
- PostgreSQL
- IntelliJ IDEA
